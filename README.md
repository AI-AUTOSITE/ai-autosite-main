## 🔒 Privacy Protection - Our Promise

**Your data never leaves your browser. Period.**

### What We DON'T Do

❌ No cookies or localStorage  
❌ No Google Analytics or tracking  
❌ No user identification or fingerprinting  
❌ No external data transmission  
❌ No email collection or sign-ups  
❌ No IP address logging  

### What We DO

✅ All processing in browser memory  
✅ Data cleared on page reload  
✅ No personal information required  
✅ Complete transparency  
✅ Open source code for verification  

**Why?** We believe tools should serve you, not track you. This site will never become a "personal information handler" under any privacy law.

## 🔐 Privacy Protection Implementation

### Decision Rule

**Does the data leave the browser or persist after reload?**
- **YES** → ❌ Absolutely Forbidden
- **NO** → ✅ OK

### Code Examples

#### ✅ GOOD - Privacy Protected
```typescript
'use client';
import { useState } from 'react';

export default function TextCounter() {
  // ✅ Memory only (cleared on reload)
  const [text, setText] = useState('');
  const [count, setCount] = useState(0);

  // ✅ All processing in browser
  const handleChange = (e: React.ChangeEvent) => {
    setText(e.target.value);
    setCount(e.target.value.length);
  };

  // ✅ Download to user's device
  const downloadResult = () => {
    const blob = new Blob([text]);
    const url = URL.createObjectURL(blob);
    const link = document.createElement('a');
    link.href = url;
    link.download = 'result.txt';
    link.click();
  };

  return (
    
      
      Count: {count}
      Download
    
  );
}
```

#### ❌ BAD - Privacy Violations
```typescript
// ❌ Persistent storage
localStorage.setItem('userData', JSON.stringify(data));

// ❌ Cookies
document.cookie = 'user=123';

// ❌ External transmission
fetch('/api/track', {
  method: 'POST',
  body: JSON.stringify(userData)
});

// ❌ Tracking scripts


// ❌ User identification
const userId = crypto.randomUUID();
```

## 🚀 Quick Start

### Prerequisites

- Node.js 18.17 or later
- npm, yarn, pnpm, or bun

### Installation
```bash
# 1. Clone the repository
git clone https://github.com/ai-autosite.git
cd ai-autosite

# 2. Install dependencies
npm install

# 3. Run development server
npm run dev

# 4. Open http://localhost:3000
```

### Common Commands

**PowerShell:**
```powershell
# Delete build folder
Remove-Item -Path ".next" -Recurse -Force

# Run dev server
npm run dev

# Install package (always specify version!)
npm install package-name@1.2.3
```

**Git Bash:**
```bash
# Delete build folder
rm -rf .next

# Run dev server
npm run dev

# Install package (always specify version!)
npm install package-name@1.2.3
```
